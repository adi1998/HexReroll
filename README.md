## Hex Reroll

Hades II mod allowing you to reroll Hexes.

## Installation

Use r2modman by ebkr from [Thunderstore](https://thunderstore.io/package/ebkr/r2modman/) or [GitHub](https://github.com/ebkr/r2modmanPlus/releases/latest).

It is recommended to backup your save from `%USERPROFILE%\Saved Games\Hades II\Profile*.sav` before modding the game in case there are issues.
